### Task 3: Изучить API стадий "Моего плана"

> "Мой план" в Битриксе использует `task.stages.*` — Kanban-стадии, привязанные к пользователю.

**Step 1: Получить текущие стадии**

```bash
WEBHOOK=$(grep BITRIX24_WEBHOOK_URL ~/bitrix24-mcp/.env | cut -d= -f2)
curl "${WEBHOOK}task.stages.get?isAdmin=0"
```
Ожидаемый результат: JSON со списком стадий. Если пусто — в следующих шагах создадим.

**Step 2: Создать стадию для теста**

```bash
curl -X POST "${WEBHOOK}task.stages.add" \
  -H "Content-Type: application/json" \
  -d '{"fields": {"TITLE": "TEST_DOMAIN", "COLOR": "FF5733", "SORT": 100}}'
```
Ожидаемый результат: `{"result": <stageId>}` — запомни ID.

**Step 3: Найти задачу для теста (или создать)**

```bash
curl "${WEBHOOK}tasks.task.list?filter[RESPONSIBLE_ID]=<userId>&select[]=ID&select[]=TITLE"
```
Ожидаемый результат: список задач, запомни любой `ID`.

**Step 4: Привязать задачу к стадии**

```bash
curl -X POST "${WEBHOOK}task.stages.movetask" \
  -H "Content-Type: application/json" \
  -d '{"id": <taskId>, "stageId": <stageId>}'
```
Ожидаемый результат: `{"result": true}` — задача теперь в стадии.

**Step 5: Убедиться в Битриксе**

Открой "Мой план" в браузере — задача должна быть в колонке `TEST_DOMAIN`.

**Step 6: Удалить тестовую стадию**

```bash
curl -X POST "${WEBHOOK}task.stages.delete" \
  -H "Content-Type: application/json" \
  -d '{"id": <stageId>}'
```

**Step 2: Создать `src/tools/planner.ts`**

```typescript
import { loadConfig } from "../config.js";

const WEBHOOK_URL = process.env.BITRIX24_WEBHOOK_URL!;

async function bitrixCall(method: string, params: Record<string, unknown> = {}): Promise<unknown> {
  const url = `${WEBHOOK_URL}${method}`;
  const response = await fetch(url, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify(params),
  });
  const data = await response.json() as { result: unknown; error?: string };
  if (data.error) throw new Error(`Bitrix error: ${data.error}`);
  return data.result;
}

async function getOrCreateStage(stageName: string): Promise<number> {
  const stages = await bitrixCall("task.stages.get", { isAdmin: 0 }) as Array<{ ID: string; TITLE: string }>;
  const existing = stages.find((s) => s.TITLE === stageName);
  if (existing) return parseInt(existing.ID);

  const stageId = await bitrixCall("task.stages.add", {
    fields: { TITLE: stageName, COLOR: "3bafda", SORT: 100 },
  }) as number;
  return stageId;
}

export const plannerTools = {
  bitrix24_add_to_my_plan: {
    description: "Добавляет задачу Битрикс24 в 'Мой план' в колонку домена воркспейса",
    inputSchema: {
      type: "object",
      properties: {
        taskId: { type: "number", description: "ID задачи в Битриксе" },
        domain: {
          type: "string",
          enum: ["dev", "finance", "marketing", "ai-architecture", "content", "legal"],
          description: "Домен воркспейса — определяет колонку в 'Моём плане'",
        },
      },
      required: ["taskId", "domain"],
    },
    async handler({ taskId, domain }: { taskId: number; domain: string }) {
      const config = loadConfig();
      const stageName = config.domainToStage[domain];
      if (!stageName) throw new Error(`Unknown domain: ${domain}`);

      const stageId = await getOrCreateStage(stageName);
      await bitrixCall("task.stages.movetask", { id: taskId, stageId });
      return { content: [{ type: "text", text: `Task ${taskId} moved to stage "${stageName}" (id: ${stageId})` }] };
    },
  },

  bitrix24_list_my_plan: {
    description: "Читает 'Мой план' из Битрикса — возвращает задачи по колонкам (доменам)",
    inputSchema: {
      type: "object",
      properties: {},
      required: [],
    },
    async handler() {
      const stages = await bitrixCall("task.stages.get", { isAdmin: 0 }) as Array<{ ID: string; TITLE: string }>;
      const config = loadConfig();
      const knownStageNames = new Set(Object.values(config.domainToStage));

      const result: Record<string, unknown[]> = {};

      for (const stage of stages) {
        if (!knownStageNames.has(stage.TITLE)) continue;
        const tasks = await bitrixCall("tasks.task.list", {
          filter: { STAGE_ID: stage.ID },
          select: ["ID", "TITLE", "STATUS", "DEADLINE"],
        }) as { tasks: unknown[] };
        result[stage.TITLE] = tasks.tasks ?? [];
      }

      return { content: [{ type: "text", text: JSON.stringify(result, null, 2) }] };
    },
  },

  bitrix24_sync_workspace: {
    description: "Синхронизирует статусы задач из Битрикса обратно — возвращает данные для обновления todos.md",
    inputSchema: {
      type: "object",
      properties: {},
      required: [],
    },
    async handler() {
      const stages = await bitrixCall("task.stages.get", { isAdmin: 0 }) as Array<{ ID: string; TITLE: string }>;
      const config = loadConfig();
      const stageNameToDomain = Object.fromEntries(
        Object.entries(config.domainToStage).map(([domain, name]) => [name, domain])
      );

      const syncData: Array<{ domain: string; stageName: string; tasks: unknown[] }> = [];

      for (const stage of stages) {
        const domain = stageNameToDomain[stage.TITLE];
        if (!domain) continue;
        const tasks = await bitrixCall("tasks.task.list", {
          filter: { STAGE_ID: stage.ID },
          select: ["ID", "TITLE", "STATUS", "DEADLINE", "CREATED_DATE"],
        }) as { tasks: unknown[] };
        syncData.push({ domain, stageName: stage.TITLE, tasks: tasks.tasks ?? [] });
      }

      const summary = syncData
        .map((d) => `## ${d.domain} (${d.stageName})\n${(d.tasks as Array<{ ID: string; TITLE: string; STATUS: string }>).map((t) => `- [${t.STATUS === "5" ? "x" : " "}] [#${t.ID}] ${t.TITLE}`).join("\n")}`)
        .join("\n\n");

      return { content: [{ type: "text", text: `Данные для обновления todos.md:\n\n${summary}` }] };
    },
  },
};
```
