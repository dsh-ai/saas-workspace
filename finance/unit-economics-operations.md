# Юнит-экономика и операционные затраты

> Последнее обновление: 2026-03-22

## 1. Себестоимость AI-обработки (COGS) на 1 активного пользователя/мес

Базовый сценарий: 10 форм, 30 ответов каждая (детализацию — в `ai-costs.md`).

| Статья расходов | Технология | Стоимость |
| :--- | :--- | :--- |
| Анализ ответов (analyze_single) | YandexGPT 5 Lite | ~180₽ |
| Батч-анализ (analyze_batch) | DeepSeek 3.2 via AI Studio | ~14₽ |
| Ретроспектива + генерация полей | Claude Haiku | ~11₽ |
| Прочие AI-вызовы | Haiku / Yandex Lite | ~5₽ |
| Хранение данных | Selectel Object Storage | ~5₽ |
| **ИТОГО COGS переменные** | | **~215₽/пользователь/мес** |

Диапазон: ~70₽ (лёгкий пользователь) — ~410₽ (активный).

---

## 2. Экономика подписки (Тариф AI — 4990₽/мес)

- **Переменные затраты (AI + хранение):** ~215₽
- **Эквайринг (3.5%):** ~175₽
- **Налоги (УСН 6%):** ~300₽
- **Валовая прибыль:** ~4 300₽ с одной подписки (~86%)
- **LTV (12 мес, без churn):** ~51 600₽

> При тяжёлом пользователе (400₽ AI): валовая прибыль ~4 115₽ (~82%) — всё равно высокая маржа.

---

## 3. Постоянные затраты (Fixed Costs)

| Ресурс | Цена/мес |
|--------|----------|
| Selectel VPS #1 (приложение, 2CPU/4GB) | ~1 500₽ |
| Selectel VPS #2 (PostHog аналитика, 2CPU/4GB) | ~1 500₽ |
| Selectel Object Storage (бэкапы) | ~50₽ |
| **Итого fixed** | **~3 050₽** |

---

## 4. Точка безубыточности

- Покрытие fixed costs: 3 050₽ / 4 300₽ = **0.71 подписки**
- Фактически: **1–2 активных AI-подписки** покрывают серверные расходы.
