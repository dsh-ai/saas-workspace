## Платёжный шлюз

**Выбран: Модульбанк интернет-эквайринг** (решение от 2026-03-29)

- Комиссия: 2,49% (до 1 млн ₽/мес), 2,39% (1–5 млн), 2,29% (5 млн+)
- Онлайн-касса: МодульКасса, ~2 500₽/мес, нативная интеграция с эквайрингом (54-ФЗ)
- Рекуррентные платежи: поддерживаются из коробки
- Выплаты: T+1 стандарт, T+0 (каждый час) за доп. опцию
- Причина выбора: р/с уже в Модульбанке → минимум интеграций, быстрый запуск

> ⚠️ Тарифы выше требуют верификации по актуальным условиям интернет-эквайринга Модульбанка — действующий PDF (`finance/интернет-эквайринг/тарифы модульбанка.pdf`) описывает только оффлайн-оборудование и ПО ModulCash, ставки интернет-эквайринга в нём нет. До получения официального тарифа цифры считаем рабочей оценкой.

### Рассмотренная альтернатива: QR/СБП (отклонена как основной способ, 2026-05-03)

Источник: https://modulbank.ru/qr-pay/

**Плюсы:**
- Комиссия ~0,4–0,7% (vs 2,49% по картам) — экономия ~80₽ с подписки 4 990₽.
- Мгновенное зачисление средств.
- Нет привязки карты — снижает барьер для части аудитории.

**Блокер для SaaS-подписки — нет рекуррентных платежей:**
- C2B QR/СБП у Модульбанка по `/qr-pay/` — разовая оплата: пользователь каждый раз сканирует QR и подтверждает в банковском приложении.
- Для подписки 4 990₽/мес = ручная оплата каждый месяц → рост churn, неприемлемо для B2B-аудитории (РОПы/владельцы отделов продаж).
- «СБП Subscription» (рекуррент через СБП) — отдельный продукт ЦБ, на 2026-05 в массе не доступен малому бизнесу.

**Решение:** карточный интернет-эквайринг — основа для подписки (рекуррент). QR/СБП — добавить вторым способом для **разовых операций**:
- покупка AI-кредитов/доп. ответов;
- апгрейд тарифа в моменте;
- опция «оплатить вручную» для тех, кто принципиально не хранит карту.

В юнит-экономике считаем по карточному эквайрингу (консервативно).
