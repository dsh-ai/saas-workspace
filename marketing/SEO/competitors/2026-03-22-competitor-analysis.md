## 4. Стратегии конкурентов в SEO

### Битрикс24 — контентный монстр
- Собственный блог с тысячами статей (`/journal/`)
- Захватывает все информационные запросы ("как составить план продаж", "контроль продаж")
- Брендирование через контент: все статьи ссылаются на продукт

### YouGile — паразитный SEO через Habr
- Активно публикует на Habr.com от имени компании
- Топовые позиции по запросам "лучшие CRM", "таск-трекеры" через чужой авторитет
- Упоминания в KP.ru, Lifehacker, Skillbox

### OkoCRM — нишевый SEO-блог
- Глубокие статьи по "контроль менеджеров", "план продаж" — нацелены на РОП
- Каждая статья = посадочная страница для продукта
- Ранжируется в ТОП-5 по запросам целевой аудитории

### Kaiten — прямой продуктовый SEO
- Страница тарифов в ТОП по "система управления задачами для бизнеса"
- Акцент на коммерческие, а не информационные запросы

## 5. Пробелы и возможности для unilist.ru

### 5.1 Незанятые ниши с коммерческим потенциалом

| Кластер | Почему интересен |
|---|---|
| **РОП-инструменты** ("инструменты РОПа", "дашборд РОПа", "чек-лист РОПа") | Нет сильных конкурентов с продуктовыми страницами под РОПа |
| **"дашборд отдела продаж"** (97/мес) | Униклист = визуальный контроль → прямая релевантность, низкая конкуренция |
| **"план факт продаж онлайн"** | Длинный хвост, коммерческая интентность, нет явного лидера |
| **"отчетность по менеджерам продаж"** | Специфический функциональный запрос, медиа его не занимают |
| **"управление b2b продажами"** (75/мес) | Нишевый, низкая конкуренция, высокое LTV-аудитории |

### 5.2 Информационные кластеры для входа через контент

Все конкуренты активно занимают:
- "как контролировать менеджеров по продажам"
- "план продаж для менеджеров"
- "KPI менеджера по продажам"

Стратегия: публиковать статьи под эти запросы с явной привязкой к продукту (как YouGile делает через Habr). Либо — размещать гостевые материалы на Habr/VC.ru.

## 6. Приоритетные действия

### Срочно (технический фундамент)
1. Проверить индексацию сайта в Яндекс.Вебмастере — похоже, проблема с индексацией
2. Убедиться, что главная страница доступна поисковому роботу
3. Настроить sitemap.xml и robots.txt

### Краткосрочно (1–2 месяца)
4. Создать посадочные страницы под кластеры:
   - `/features/dashboard` → "дашборд отдела продаж"
   - `/features/sales-control` → "контроль менеджеров продаж"
   - `/rop` → инструменты для руководителя отдела продаж
5. Написать 3–5 статей в блог:
   - "Дашборд РОПа: что должно быть на экране руководителя продаж"
   - "Контроль менеджеров без CRM: 5 инструментов"
   - "Как отслеживать план-факт продаж в реальном времени"

### Среднесрочно (3–6 месяцев)
6. Гостевые публикации на Habr.com (под аккаунтом компании)
7. Попасть в рейтинги KP.ru, Lifehacker, workspace.ru
8. SEO-оптимизация главной страницы под запросы "управление отделом продаж"

## 7. Ключевые выводы

1. **Unilist.ru = нулевая видимость.** Сайт не ранжируется ни по одному запросу, включая брендовые. Это первопричина отсутствия SEO-трафика.

2. **Рынок занят, но есть щели.** Общие запросы (CRM, таск-трекер) захвачены Битрикс24, amoCRM. Но РОП-специфичные запросы и "дашборд продаж" практически свободны.

3. **Контент = единственный быстрый путь.** Продуктовые страницы дают позиции медленно. Контент под информационные запросы РОПа — более быстрый вход.

4. **YouGile = модель для подражания.** Они захватили позиции не через собственный домен, а через Habr. Та же стратегия доступна unilist.ru без SEO-фундамента.
