# Лендинг: Отделы продаж B2B-компаний с Bitrix24

---

## Oneliner

**Unilist** — квалифицируй лидов через анкету и получай ответы прямо в карточке Bitrix24, чтобы менеджеры тратили время на продажи, а не на ввод данных.

---

## Core Job + ценность + фичи

Каждый лид — через единую анкету. Ответы — сразу в Bitrix24. Без звонков «чтобы уточнить», без ручного ввода, без потерянного контекста.

Unilist создаёт профессиональные формы, которые клиент открывает по ссылке с телефона, а менеджер получает готовый профиль в карточке сделки — до первого звонка.

**За счёт:**
- Публичных форм по ссылке — клиент заполняет без регистрации
- Нативной интеграции с Bitrix24 — ответы появляются в карточке сделки автоматически
- Уведомлений в реальном времени — менеджер звонит, пока клиент ещё «горячий»
- Конструктора без разработчика — первая форма за 15 минут

**Micro Jobs:**
- Квалифицировать нового лида через структурированную анкету — по ссылке, без регистрации
- Автоматически получить ответы клиента в карточке Bitrix24
- Узнать потребности клиента до встречи — через форму-бриф, отправленную заранее

## Доносим ценность

- **Данные в Bitrix24 без ввода** — менеджер тратит ноль минут на занесение данных клиента. Информация появляется в карточке автоматически сразу после заполнения формы.
- **Встреча начинается с решений** — клиент уже ответил на базовые вопросы через форму. Первый контакт — про ценность, не про «расскажите о себе».
- **Звонишь по горячему следу** — уведомление приходит в момент заполнения. Менеджер звонит через 5–10 минут — конверсия из такого звонка кратно выше, чем из холодного.

## Человек узнаёт себя

### Узнаёшь себя?

**Ты хочешь выстроить процесс, который работает без тебя?**
Или **ты хочешь наконец перестать объяснять собственнику, почему лиды «теряются»?**

- После каждой планёрки думаешь: «опять половина менеджеров ничего не внесла в CRM»?
- Собственник спрашивает конверсию по этапам — а у тебя нет данных, только ощущения?
- Новый менеджер вышел — и снова весь контекст по клиентам у него в голове, не в системе?
- Клиент приходит на встречу, а менеджер первые 20 минут тратит на базовые вопросы?
- Тревожно, что ключевой менеджер уйдёт — и половина истории по клиентам уйдёт с ним?
- Злишься, когда понимаешь: ты не можешь масштабировать отдел, потому что каждый работает «по-своему»?

## Объясняем, как ты сможешь выполнить Core Job

### Как квалифицировать лидов системно и без ручного труда?

- **Отправить анкету по ссылке →** Клиент заполняет за 3–5 минут с телефона, без регистрации. Ты получаешь структурированный профиль ещё до первого звонка — и знаешь, стоит ли этот лид твоего времени.
- **Ответы автоматически в Bitrix24 →** Менеджер не вводит ничего вручную. Данные появляются в карточке сделки сразу. Ошибок нет, время сэкономлено, история полная.
- **Бриф перед встречей →** Клиент заранее отвечает на ключевые вопросы. Первая встреча начинается с предложения, а не с выяснения потребностей. Конверсия растёт уже на этапе презентации.
- **Уведомление в момент заполнения →** Менеджер звонит через 5–10 минут — клиент ещё «в контексте». Это принципиально другой разговор, чем звонок через день.
- **Опрос NPS после сделки →** Клиент получает форму автоматически при смене статуса. Данные об удовлетворённости копятся без административных усилий — раз в квартал видишь тренд.

## Загружаем точку Б

### Ты почувствуешь:

- **Спокойствие** — когда на планёрке видишь: каждый лид прошёл одинаковую квалификацию, все данные в Bitrix24, ни одного «я не успел внести»
- **Уверенность** — когда собственник спрашивает про конверсию и ты открываешь дашборд с реальными данными за квартал, а не импровизируешь
- **Контроль** — когда новый менеджер выходит на работу и с первого дня работает по процессу, который ты выстроил — процесс больше не зависит от конкретного человека

### Ты выполнишь:

- **Построить систему, которая работает без тебя** — квалификация лидов, сбор данных, NPS — всё идёт по единому процессу, независимо от того, кто конкретно ведёт сделку
- **Стать РОПом, которому собственник доверяет** — у тебя есть данные, а не ощущения. Ты приходишь на встречу с цифрами, а не с историями о «сложном рынке»
- **Масштабировать отдел без потери качества** — новый менеджер получает готовый процесс: форма → Bitrix24 → уведомление → звонок. Стандарт работает для всех одинаково

## Снижаем барьеры

- **«У нас уже есть Bitrix24, зачем ещё один инструмент?»**
  Unilist работает *внутри* Bitrix24: ответы клиентов появляются прямо в карточке сделки. Это не замена CRM — это форма, которой в Bitrix24 никогда не было. Ты не меняешь систему, ты добавляешь недостающий блок.

- **«Некогда разбираться, у нас и так всё занято»**
  Первая форма — за 15 минут. Подключить Bitrix24 — ещё 5 минут. Шаблон квалификационной анкеты уже есть — просто адаптируй под свой продукт. Отправь первому лиду сегодня.

- **«Клиенты не будут заполнять анкеты»**
  Форма открывается по ссылке без регистрации, выглядит профессионально, работает с любого телефона. Клиент видит не «очередную гугл-форму», а структурированный запрос от компании, которой он уже доверяет. Средний процент заполнения — выше, чем ответ на звонок.

## Увольняем конкурентов

- **Google Forms**
  Нет интеграции с Bitrix24. Нет уведомлений в реальном времени. Выглядит как анкета в школе — не как профессиональный контакт с клиентом B2B. Данные оседают в таблице, которую никто не смотрит.

- **Typeform**
  Нет Bitrix24. Нет русскоязычного интерфейса. Дорогой — платишь за красоту, не за результат в CRM. Клиент B2B-сегмента с трудом понимает иностранный сервис.

- **Кастомная разработка или интеграция через разработчика**
  2–4 недели минимум, бюджет от 50–150 тысяч, потом нужна поддержка при каждом изменении. Unilist — 15 минут, изменения формы сам, без тикетов в IT.
