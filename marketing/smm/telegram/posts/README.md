# Posts

Черновики постов для Telegram-канала «Продажи без тёмных пятен».

## Структура

| Папка | Что внутри |
|---|---|
| `drafts/` | Черновики, сгенерированные Claude. Имя файла = `<id>.md`. Бот подхватит автоматически. |
| `scheduled/` | (резерв, не используется в MVP — статус хранится в БД на VPS) |
| `published/` | (резерв, не используется в MVP — после публикации файл остаётся в `drafts/`) |

## Workflow

1. Claude пишет в `drafts/`
2. `git push` → cron на VPS (каждые 5 мин) делает `git pull` + `cp -n`
3. Бот раз в минуту сканирует, шлёт владельцу approve-карточку за 24ч до `publish_at`
4. На approve — публикует в канал

Подробности в `../CLAUDE.md` (раздел «Генерация постов для бота»).

## Формат файла

См. шаблон в `../strategy.md` или пример:

```markdown
---
id: 2026-05-12-1000-lost-deal
publish_at: 2026-05-12T10:00:00+03:00
pillar: pain
format: text
---

Вчера потерял сделку на 1.2 млн. Виной — опросный лист.

[тело поста...]

CTA: Подпишись, если работаешь с длинным циклом B2B.
```
