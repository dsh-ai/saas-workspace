# Фича: Guided Voice Capture

**Дата:** 2026-03-28
**Статус:** Design in progress (отложено, не приоритет в моменте)

---

## Идея

Продолжение Voice Input. Вместо свободного голосового ввода по полям — управляемый режим: система последовательно предлагает темы, пользователь проговаривает ответ, AI структурирует и заполняет поле.

---

## Ключевые решения (принятые)

- **Режим:** альтернатива ручному вводу, не замена. Оба режима доступны, пользователь переключается.
- **Вопросы:** только текст на экране, без TTS. Тема берётся автоматически из лейбла поля — без дополнительной настройки создателем.
- **Обработка голоса:** AI структурирует ответ → пользователь видит результат и подтверждает/редактирует → переход к следующему полю.
- **Прерывание:** в любой момент, уже заполненные поля сохраняются.

---

## Рассмотренные форматы UI (не выбран)

**А. Оверлей-мастер** — полноэкранный оверлей, по одному вопросу за раз. Чистый фокус, хорошо на мобильных. *Рекомендован как отправная точка.*

**Б. Встроенная панель-гид** — панель сбоку/сверху, форма видна. Сложнее на мобильных.

**В. Чат-интерфейс** — заменяет форму на диалоговый UI. Наибольшая сложность реализации.

Формат взаимодействия требует дополнительной продуктовой проработки.

---

## Технические зависимости

- Web Speech API (как в voice-input)
- Бэкенд: LLM-вызов для структурирования ответа (отличие от voice-input, где всё на клиенте)
- Доступность: AI-план (4990₽/мес)
