# Технологический стек и AI-контур

## 1. Архитектура обработки данных
1.  **Загрузка:** Аудиофайл (MP3/WAV) через API на сервер.
2.  **Обработка звука:** FFmpeg (нормализация громкости, удаление шумов).
3.  **STT (Speech-to-Text):**
    *   *Основной:* Yandex SpeechKit (лучшая поддержка русского языка).
    *   *Резервный:* OpenAI Whisper (требует GPU-серверов в РФ).
4.  **LLM (Логика):**
    *   *Вариант А (Облачный):* GigaChat API или YandexGPT API. Полная юридическая чистота в РФ.
    *   *Вариант Б (Self-hosted):* Llama 3 / Saiga на своих мощностях (высокие затраты на железо, максимальная приватность).

## 2. Технологический стек
*   **Backend:** Python 3.10+, FastAPI.
*   **Database:** PostgreSQL (основная), Redis (очереди задач на обработку аудио).
*   **Task Queue:** Celery (для фоновой расшифровки длинных файлов).
*   **Storage:** S3-совместимое хранилище (для аудиозаписей).
